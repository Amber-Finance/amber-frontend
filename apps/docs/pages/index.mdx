# Amber Finance Documentation

Welcome to the Amber Finance documentation. Here you'll find everything you need to know about using our platform.

## Getting Started

Amber Finance is a decentralized finance (DeFi) platform built for Bitcoin. Our platform allows you to:

- **Lend** your Bitcoin and earn yield
- **Borrow** assets using your Bitcoin as collateral
- **Trade** various Bitcoin-backed assets
