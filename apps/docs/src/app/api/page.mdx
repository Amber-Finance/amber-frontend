## REST API

Amber Finance will provide a REST API for accessing market data, user positions, and other protocol information.

### Endpoints

- `/api/markets` - Get all available markets
- `/api/markets/{id}` - Get specific market information
- `/api/user/{address}` - Get user positions and balances

## GraphQL API

A GraphQL API will also be available for more flexible data querying.

## Rate Limits

API endpoints will have rate limits to ensure fair usage across all users.

## Authentication

Most endpoints are public and don't require authentication. For user-specific data, wallet signatures may be required.
