### Usage Guidelines

- **Icon**: Use the icon for favicons, small applications, and compact spaces
- **Icon Grids**: Icon grid variations for UI/UX design and reference
- **Logo Claim**: Use logos with tagline for official communications and branding
- **Logo Simple**: Use simple logos without tagline for compact spaces and technical applications
- **Logo Grids**: Logo grid variations for comprehensive brand reference
- **Format**: Prefer SVG formats for web use, PNG for print materials where transparency is needed
